MarkDown說明（引自Wiki）

Markdown 是一種輕量級標記式語言，創始人為約翰·格魯伯（John Gruber）。它允許人們「使用易讀易寫的純文字格式編寫文件，然後轉換成有效的XHTML(或者HTML)文件」。[4]這種語言吸收了很多在電子郵件中已有的純文字標記的特性。wiki

###

使用原因

  1. 簡單、輕型
  2. github支援
  3. 電子書使用
  4. 排版整齊漂亮
